# Supermarket Sales Analysis

Analyzing 1,000 transactions across 3 different supermarket branches in a 3 month time period.
Project created using SQL and Tableau. 

---

## Key Findings

- Peak sales occur between 6-8pm
- Females outspend males in all product categories except Health & Beauty
- Members outspend normal customers by about $56k dollars
- Saturdays and Tuesdays are the 2 biggest days in sales, while Monday is the lowest.
- The Cairo store shows more hourly sales volatility compared to the other 2 stores.
